# Semantic segmentation of plant organs using Encoder/Decoder network and RGB-D images


This project is about semantic segmentation of 4 classes of plant organs (stem, petiole, leaf and fruit) from RGB-D images using Pyhton 3, Tensorflow 2.0 and Keras. I worked with synthetic data, randomly generated and annotated with Blender and used an Encoder/Decoder architecture.
